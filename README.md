盗火者计划。
老年人受限于认知惯性与信息茧房，非常容易被鸡汤和非科学的短视频、文章洗脑。
如果你也尝试过把科学的视频、文章发给他们，他们却有各种各样借口，看似自洽、实际总有很多漏洞的逻辑来为自己辩白，不愿接受，只愿意相信自己想相信的。
